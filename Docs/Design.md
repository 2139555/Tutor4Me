- Student
- Tutor
- System/Admin

## <u>User Stories</u>
- ### <u>Student</u>
    - User/Student
        - Student registration
        - Student login
        - Update student
        - Delete student

    - Tutor
        - View tutor by city
        - View tutor by time slot
        - View tutor by subject
        - View all tutors
        - View available tutoring slot for a tutor
        - Book a tutor

    - Rating
        - Add tutor rating
        - View tutor rating

- ### <u>Tutor</u>
    - User/Tutor
        - Tutor registration
        - Tutor login
        - Update tutor
        - Delete tutor

    - Student
        - View student details for the student that booked a time slot

    - Time Slot
        - Create time slot
        - View time slot
        - Update time slot
        - Delete time slot

    - OfferedSubject
        - Add subject offered
        - View subject offered
        - Remove subject offered

    - Rating
        - View own rating

- ### <u>System/Admin</u>
    - Role
        - Create role
        - View role
        - Update role
        - Delete role

    - City
        - Create city
        - View city
        - Update city
        - Delete city

    - Subject
        - Create subject
        - View subject
        - Update subject
        - Delete subject
